friend was created by Kenan Poole as a response to
The Garden of Forking Paths by Jorge Luis Borges.


I want to use this website as an example how to make an interactive website without javascript. Not only is it possible to do so, but the folder for this website is similar to the book described in The Garden of Forking Paths. The labyrinth novel from the short story was full of contradictories and out of order. In the same sense the name of the html pages are numbered but it would not make sense to read each html page in sequential order. There is a document that lists all the paths between the html pages clarifying the mess of a folder.
